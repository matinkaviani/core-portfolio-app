# Affiliate System

## Overview
A platform connecting advertisers and publishers with tracking and analytics.

## Responsibilities
- Built dashboards for publishers and advertisers
- Implemented tracking UI components
- Designed data visualization layers

## Tech Stack
- Next.js
- TypeScript
- REST APIs

## Key Challenges
- Data consistency across dashboards
- Real-time analytics representation
- Complex UI state management

## Outcome
Unified analytics dashboards with consistent tracking across publisher and advertiser views.
