# Dara (CSDI Iran)

## Overview
Unified stakeholder portal for Iran's capital market under the Central Securities Depository of Iran (CSDI) — real-time portfolio data, transaction reports, dividend history, and e-voting for general assemblies.

## Responsibilities
- Built investor, issuer, and broker-facing modules with role-appropriate data access
- Implemented high-density tables and reporting views with TanStack Table
- Designed form-heavy workflows with React Hook Form and server-backed validation
- Shipped accessible UI with Shadcn UI and Tailwind CSS

## Tech Stack
- Next.js
- TypeScript
- Shadcn UI
- Tailwind CSS
- React Hook Form
- React Query
- TanStack Table

## Key Challenges
- Presenting regulated financial data clearly across multiple stakeholder types
- Large tabular datasets with filtering, sorting, and export expectations
- Reliable data fetching patterns for portfolio and transaction reporting

## Outcome
Production portal connecting capital-market stakeholders to real-time holdings, reports, and assembly voting workflows.
