# Agah Research

## Overview
Core frontend modules for Agah Broker's stock research and market data analysis platform — reusable UI, complex tables, and interactive charts for institutional-grade market visualization.

## Responsibilities
- Built research workflows, data tables, and chart-driven analysis views
- Designed state architecture with Recoil and Context API to manage complex market data flows
- Collaborated in a 9-engineer team on component integration and release quality
- Migrated the stack to Next.js and TypeScript for better maintainability and performance

## Tech Stack
- React
- Next.js
- TypeScript
- Highcharts
- Ant Design
- Recoil

## Key Challenges
- Rendering intricate market charts without sacrificing readability
- Managing deeply nested component state across research modules
- Balancing data density with responsive, analyst-friendly layouts

## Outcome
Production research platform with improved statistical readability, faster page loads, and a more maintainable Next.js architecture.
