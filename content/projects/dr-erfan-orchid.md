# Dr Erfan Orchid

## Overview
Multi-tenant e-commerce SaaS with a responsive storefront, customizable product flows, advanced checkout, and dynamic shipping logic — built for scale across tenants.

## Responsibilities
- Architected the storefront and tenant-facing commerce flows on Next.js
- Implemented order management, inventory tracking, and configurable shipping rules
- Delivered accessible, polished shopping UX with Shadcn UI and Tailwind CSS
- Structured forms and validation with React Hook Form and Zod

## Tech Stack
- Next.js
- TypeScript
- Shadcn UI
- Tailwind CSS
- React Hook Form
- Zod
- Payload CMS
- React Query
- Monorepo

## Key Challenges
- Multi-tenant data isolation and per-tenant configuration
- Complex checkout and shipping rule composition
- Consistent design system across admin and storefront surfaces

## Outcome
Live SaaS commerce platform with tenant-ready operations, responsive storefront, and maintainable monorepo structure.
