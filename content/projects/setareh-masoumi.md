# Setareh Masoumi

## Overview
High-performance personal portfolio with smooth motion design, SEO-optimized routing, and a Strapi-backed CMS for fully editable galleries, sections, and blog content.

## Responsibilities
- Designed and implemented the frontend architecture on Next.js
- Built animation-driven layouts with Framer Motion without sacrificing performance
- Integrated Strapi for dynamic content management across site sections
- Optimized routing and metadata for search visibility

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Strapi

## Key Challenges
- Balancing rich motion with Core Web Vitals and perceived performance
- Flexible CMS schema for galleries, blog posts, and editable page sections
- Clean content-to-UI mapping for non-technical editors

## Outcome
Live portfolio site with polished motion, editor-friendly CMS workflows, and SEO-ready page structure.
